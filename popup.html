Extension Information

Keyboard Shortcuts

  • Ctrl + S - Save form
  • Ctrl + C - Copy report
  • Esc - Close dropdowns/overlay

Utility Buttons

  • ℹ️ (i) - Show this information
  • ⤢ - Pop out to separate window
  • ⚙️- Manage permissions
  • ↻ - Refresh permissions and features

Auto-Save Feature

  • Your form data is automatically saved while typing
  • 'Last saved' indicator shows when your changes were last saved
  • If you see 'Unsaved changes', wait a moment for auto-save to complete

Required Permissions

  • Active Tab - For environment detection
  • Downloads - For saving screenshots
  • Scripting - For environment detection
  • Clipboard Write - For copying reports
